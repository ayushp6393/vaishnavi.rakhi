## 🌸 About Vaishnavi Rakhi

**Vaishnavi Rakhi** is a curated collection of handcrafted rakhis rooted in tradition, devotion, and artistry. Each piece is thoughtfully designed to honor the timeless bond shared between siblings during the auspicious festival of **Raksha Bandhan**.

Inspired by Vaishnav heritage and culture, our rakhis blend spiritual motifs with elegant craftsmanship — making every thread a blessing and every knot a promise.

---

## ✨ Features

- 🧵 **Handcrafted Designs** — Artisan-made with premium threads, beads, and embellishments
- 🪷 **Vaishnav-Inspired Motifs** — Tulsi beads, Shankha, Chakra, and floral patterns
- 🎁 **Gift-Ready Packaging** — Beautifully boxed for gifting
- 🛒 **Easy Ordering** — Simple catalog browsing and order placement
- 🚚 **Pan-India Delivery** — Delivered to your doorstep before the festival
- 💬 **Custom Orders** — Personalized rakhis available on request

## 🚀 Getting Started

### Prerequisites

- A modern web browser
- [Git](https://git-scm.com/) installed
- (Optional) [Node.js](https://nodejs.org/) for local development server

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/vaishnavi-rakhi.git

# Navigate into the project
cd vaishnavi-rakhi

# Open in browser (or use Live Server in VS Code)
open index.html
```

## 🛍️ How to Order

1. Browse the **catalog** and choose your favorite rakhi(s)
2. Fill in the **order form** on the Contact page
3. Make payment via **UPI / Bank Transfer** (details on order confirmation)
4. Receive a **confirmation message** with delivery timeline
5. 🎉 Your rakhi arrives gift-wrapped at your doorstep!
